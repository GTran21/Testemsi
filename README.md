# Lemon Store Simulator

This repository contains a clicker game where you click to get lemons and spend them to get more lemons and site customization features.

## What is included

- `index.html` — main website layout and interactive simulation.
- `styles.css` — responsive styling with nature-inspired colors.
- `script.js` — interaction logic for animated tree display, site customization, and upgrade purchases.

## How to use

Open the most recently deployed version of the website this repository codes for on GitHub pages to play Lemon Store Simulator.
