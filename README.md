# Lemon Tree Growth Simulator

This repository contains a small interactive website that demonstrates the typical growth stages of a lemon tree.

## What is included

- `index.html` — main website layout and interactive simulation.
- `styles.css` — responsive styling with nature-inspired colors.
- `script.js` — interaction logic for stage selection, play/pause simulation, and animated tree display.

## How to use

Open `index.html` in a browser to explore the simulation. Use the stage buttons, progress slider, and play controls to watch how the tree advances from seed to fruiting.
